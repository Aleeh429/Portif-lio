Jogo de corrida

Este projeto é um jogo de corrida simples e interativo, desenvolvido utilizando apenas HTML, CSS e JavaScript puro, sem o uso de frameworks ou bibliotecas externas. O objetivo principal do jogo é proporcionar uma experiência divertida, com uma interface responsiva e uma jogabilidade leve. O jogador controla um carro (ou outro personagem) desviando de obstáculos e tentando alcançar a maior pontuação possível.
